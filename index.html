Proyectos

ZER0
Gestor de alias de comandos para Arch Linux con REPL interactivo y +80 atajos por defecto.

Inicio rápido

¿Primera vez? Sigue estos pasos para empezar con ZER0:

1

Instala ZER0

Ejecuta el instalador de un solo archivo:

bash <(curl -fsSL https://raw.githubusercontent.com/LogLabsHQ/ZER0/main/install.sh)
2

Abre el REPL

Inicia ZER0 desde la terminal con cualquiera de estos comandos:

zero    # o también: -Z  ·  $Z
3

Explora los comandos

Revisa la referencia completa de comandos para sacar el máximo provecho a ZER0.

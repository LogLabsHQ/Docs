Comandos / Referencia

Referencia completa de todos los comandos del REPL de ZER0, organizados por categoría. ZER0 incluye más de 80 atajos por defecto.

Uso
Todos estos comandos se ejecutan dentro del REPL de ZER0. Abre el REPL con zero, -Z o $Z y luego escribe el alias.

Comandos del sistema ZER0

AliasDescripción
helpMuestra la ayuda del REPL y lista todos los comandos disponibles.
versionMuestra la versión actual de ZER0.
listLista todos los alias definidos (incluyendo personalizados).
add <alias> <cmd>Agrega un nuevo alias personalizado.
remove <alias>Elimina un alias personalizado.
editAbre el archivo de configuración en el editor por defecto.
reloadRecarga la configuración sin salir del REPL.
lang <es|en>Cambia el idioma del REPL en tiempo real.
clearLimpia la pantalla del REPL.
exit / qSale del REPL de ZER0.

Commands / Reference

Full reference of all ZER0 REPL commands, organized by category. ZER0 includes 80+ default shortcuts.

Usage
All these commands are run inside the ZER0 REPL. Open the REPL with zero, -Z, or $Z, then type the alias.

ZER0 System Commands

AliasDescription
helpShows REPL help and lists all available commands.
versionShows the current ZER0 version.
listLists all defined aliases (including custom ones).
add <alias> <cmd>Adds a new custom alias.
remove <alias>Removes a custom alias.
editOpens the config file in the default editor.
reloadReloads the configuration without leaving the REPL.
lang <es|en>Changes the REPL language in real time.
clearClears the REPL screen.
exit / qExits the ZER0 REPL.

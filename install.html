Instalación

ZER0 se instala con un solo comando. El instalador descarga y configura todo automáticamente, incluyendo Python embebido si es necesario.

Requisitos

  • Arch Linux (o derivadas: Manjaro, EndeavourOS, etc.)
  • bash — viene por defecto en Arch
  • curl — para descargar el instalador
  • Conexión a Internet
Nota
ZER0 está diseñado específicamente para Arch Linux. No se garantiza soporte oficial en otras distribuciones.

Instalación recomendada

Ejecuta este comando en tu terminal para instalar ZER0 automáticamente:

bash <(curl -fsSL https://raw.githubusercontent.com/LogLabsHQ/ZER0/main/install.sh)

El script hará lo siguiente:

  1. Verificar los requisitos del sistema
  2. Descargar ZER0 y sus dependencias
  3. Configurar los alias de terminal (zero, -Z, $Z)
  4. Crear el archivo de configuración en ~/.config/zer0/

Instalación manual

Si prefieres instalar manualmente, clona el repositorio y ejecuta el instalador localmente:

# 1. Clonar el repositorio
git clone https://github.com/LogLabsHQ/ZER0.git
cd ZER0

# 2. Ejecutar el instalador
bash install.sh

Verificar la instalación

Una vez instalado, verifica que ZER0 funciona correctamente:

zero --version

Deberías ver la salida:

ZER0 v1.0.3 — LogLabs

Primer lanzamiento

Inicia el REPL interactivo con cualquiera de estos comandos:

zero    # Nombre completo
-Z      # Atajo corto
$Z      # Variante con prefijo $
Tip
Agrega alias zero='zero' a tu ~/.bashrc o ~/.zshrc si el comando no se reconoce tras la instalación. El instalador lo hace automáticamente, pero puede requerir reiniciar la sesión de terminal.

Desinstalación

Para desinstalar ZER0 completamente:

zero --uninstall

O manualmente eliminando los archivos:

rm -rf ~/.config/zer0
rm -rf ~/.local/share/zer0
# Elimina también las líneas de alias en ~/.bashrc o ~/.zshrc

Installation

ZER0 installs with a single command. The installer downloads and configures everything automatically, including embedded Python if needed.

Requirements

  • Arch Linux (or derivatives: Manjaro, EndeavourOS, etc.)
  • bash — comes by default on Arch
  • curl — to download the installer
  • Internet connection
Note
ZER0 is designed specifically for Arch Linux. Official support on other distributions is not guaranteed.

Recommended Installation

Run this command in your terminal to install ZER0 automatically:

bash <(curl -fsSL https://raw.githubusercontent.com/LogLabsHQ/ZER0/main/install.sh)

The script will:

  1. Verify system requirements
  2. Download ZER0 and its dependencies
  3. Configure terminal aliases (zero, -Z, $Z)
  4. Create the config file at ~/.config/zer0/

Manual Installation

If you prefer to install manually, clone the repository and run the installer locally:

# 1. Clone the repository
git clone https://github.com/LogLabsHQ/ZER0.git
cd ZER0

# 2. Run the installer
bash install.sh

Verify Installation

Once installed, verify ZER0 works correctly:

zero --version

You should see:

ZER0 v1.0.3 — LogLabs

First Launch

Start the interactive REPL with any of these commands:

zero    # Full name
-Z      # Short shortcut
$Z      # Variant with $ prefix
Tip
Add alias zero='zero' to your ~/.bashrc or ~/.zshrc if the command isn't recognized after installation. The installer does this automatically, but it may require restarting your terminal session.

Uninstallation

To completely uninstall ZER0:

zero --uninstall

Or manually by removing the files:

rm -rf ~/.config/zer0
rm -rf ~/.local/share/zer0
# Also remove alias lines from ~/.bashrc or ~/.zshrc
